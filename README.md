# Temnaya Zashelina

This is a bad place if people died here. The wrong way.

## Description

A game where the end is always near.

## Features

- Dangerous mobs
- Scary SFX
- Character customization
- Encounter others and play with friends
- OST
- Multiple procedurally generated biomes
- Branched story

## Getting Started

1. **Installation:**
   - Download from contentDB, if possible
   - or
   - Clone the repository

2. **Configuration:**
   - Adjust game settings in `game.conf` to tailor the experience to your liking.
   

3. **Gameplay:**
   - Optionally, follow in-game tutorial.
   - Try different things to see results.
